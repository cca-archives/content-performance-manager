GOV.UK Content Performance API v1.0.0

GOV.UK Content Performance API provides metrics about GOV.UK content.

This API accepts HTTP requests and responds with JSON.

Base URL

All requests to this API should be prefixed with the following URL:

https://content-performance-manager.publishing.service.gov.uk/api/

Metrics

GET /metrics/{content_id}

Get metric values for a single content item over a date range.

Returns an array of values. Each item represents a metric value at a single point in time.

This request does not require authentication.

Parameters

Parameter In Type Required Description
content_id path string true A GOV.UK content ID. This is a UUID value as used in the Content API.
from query date true From date.
to query date true To date.

Example Request

curl 'https://content-performance-manager.publishing.service.gov.uk/api/v1/metrics/f838c22a-b2aa-49be-bd95-153f593293a3?from=2018-02-27&to=2018-02-27&metric=pageviews'

Responses

Status Meaning Description Schema
200 OK An array of items where each item represents a value of the metric at a single point in time. The array may be empty. None
400 Bad Request The API request is invalid. None
